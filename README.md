# Python零基础语法学习仓库
## 项目简介
本仓库存放Python入门全套课堂示例代码与课后练习题，覆盖变量、数据类型、运算符、字符串、分支判断等全部基础语法，是后续学习数据分析、Pytorch大模型的前置基础练习。
本人为深圳大学计算机科学与技术（卓越班）大二学生，目标保研本校大模型方向，此仓库记录完整Python自学全过程。

## 运行环境
1. Python版本：Python 3.10 及以上
2. 代码编辑器：VS Code
3. 运行方式：右键对应py文件 → 在终端中运行Python文件
4. 交互类代码（含input输入）：务必在【终端】面板输入内容，输出面板无法交互

## 整体学习路线规划
1. 第一阶段：Python基础语法（当前仓库全部代码）
2. 第二阶段：Numpy、Pandas 数据分析库
3. 第三阶段：Pytorch深度学习基础
4. 第四阶段：大模型微调、AI项目实战

## 使用方法
1. 将本仓库克隆至本地电脑
2. 使用VS Code打开 python-basic 总文件夹
3. 选择对应天数的代码文件运行、复习
4. 所有代码附带注释，标注对应课堂知识点，便于复盘复习
